# StarWars-finder-app
Android app based on swapi.dev

- Modularization
- Single Activity
- Clean Architecture
- Room
- Koin
- Coroutines, Flow
- Material3 Design
- Gradle.build on Kotlin / BuildSrc

App have three screens: Search, Favorite and Detailed screen
